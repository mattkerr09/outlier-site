What this is, honestly

Outlier is a desktop app that downloads and runs Qwen language models (Alibaba, Apache 2.0) locally on Apple Silicon Macs. It's a clean Mac-native wrapper around mlx-lm with offline-first defaults, honest tier labels, and no hype.

I'm Matt, a solo founder building in public. 18 days in, $0 revenue, three patents filed on a ternary Mixture-of-Experts architecture that's still in research. The shipping app is simpler: curated Qwen models, sensible defaults, a UI that doesn't get in the way.

If you want a local AI app that tells you exactly what it's loading and stops pretending it invented the model inside — this is the one.

What I won't do.

Some operating principles, so you know where I stand before you install anything.

No telemetry

The app doesn't phone home. No analytics, no error reporting pipeline, no "anonymous usage." If something breaks, open an issue.

No cloud fallback

If your Mac can't run the model, the app says so. It doesn't quietly ship your prompt to someone else's server.

No benchmark theater

Numbers shown are measured on the exact checkpoint that ships. No MMLU claims lifted from a related model. No "frontier-quality" without a citation.

No dark patterns

No "unlock" modals on features the model can already do. No drip-fed context windows. No pay-to-remove-watermark.
