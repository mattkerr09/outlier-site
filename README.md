# Outlier Static Site

This is the static marketing site for Outlier, focused on ternary MoE models and runtime on consumer hardware.

Serve locally with:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

Deployment:

- GitHub Pages can serve this repo directly after you enable Pages in the repository settings.
- Any static host can deploy it by serving `index.html` as the site root.
